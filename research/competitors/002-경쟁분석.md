## 2. 주요 경쟁사 상세 분석

### 2-1. 하루콩 (HaruBean) — **가장 강력한 직접 경쟁자**

| 항목 | 내용 |
|------|------|
| 개발사 | 블루시그넘 |
| 출시 | 2021년 |
| 누적 DL | 글로벌 1,000만+ (한국 ~200만, 해외 80%+) |
| 투자 | 누적 21억 원 |
| 수상 | 구글플레이 '올해의 앱' 2년 연속 |

**핵심 기능:**
- 콩 캐릭터 16종으로 감정 기록 (복수 선택 가능)
- 월별 캘린더로 감정 흐름 시각화
- 기록 블럭: 활동/수면/운동/월경 추적
- 감정 인사이트: 활동-감정 상관관계 분석 (기본적 수준)
- 캐릭터 테마 스토어 (디즈니 콜라보)

**수익 모델:** 구독 (광고 제거 + 프리미엄), 캐릭터/테마 판매, 무료 사용자 광고

**강점:**
- 압도적 설치 기반 (1,000만+)
- 직관적 UX, 귀여운 캐릭터 → 10대~20대 강력 어필
- 글로벌 확장 성공 (236개국)
- 투자 유치로 지속적 기능 확장 여력

**약점:**
- 감정 기록 '이후' 단계 부족: 분석은 기초적, 행동 제안 없음
- 한국어 특화 감정 어휘 미흡 (글로벌 전략으로 범용성 우선)
- 크로스 디바이스 동기화 미지원 (수동 백업)
- 데이터 내보내기 불가

**마음날씨 대비 위협:**
- 하루콩이 AI 패턴 분석/행동 제안 기능을 추가하면 직접적 위협
- 다만 글로벌 전략으로 한국어 특화에 집중할 가능성 낮음
- 캐릭터/감성 중심 vs 분석/행동 중심의 포지셔닝 차이

### 2-2. 루빗 (Roubit) — **습관+감정 통합 앱**

| 항목 | 내용 |
|------|------|
| 개발사 | 루빗 (대표 이준영, 2021년 대학생 창업) |
| 누적 DL | **210만+** |
| 사용자 구성 | Z세대 70%, 직원 9명 |
| Google Play 평점 | 4.48/5.0 (14,000건) |
| 핵심 특징 | 습관 + 감정 통합, ChatGPT 기반 멘탈케어, 게이미피케이션 |

**핵심 기능:**
- 루틴/할일 관리 (요일별, 카테고리별)
- 아기 루빗(토끼 캐릭터) 육성 — 루틴 완료 시 성장
- AI 위로의 편지 (ChatGPT 기반, 감정 일기 분석 후 위로 편지)
- AI 습관코치 챗봇 '에디' (습관 기록 기반 조언, 추천, 응원)
- NFC 태그 기반 강제 기록 기능 (프리미엄)

**수익 모델:** 연간 약 $40/년 (~53,000원), 무료 버전은 루틴 8개 제한

**강점:**
- 210만+ DL로 시장 검증 완료, Z세대 70% 점유
- ChatGPT 기반 AI 위로+습관코치로 감정 영역까지 커버
- 게이미피케이션(캐릭터 육성)으로 리텐션 확보
- 구글플레이 '올해를 빛낸 자기계발 앱' 수상

**약점:**
- 습관 관리가 주축 → 감정 분석의 깊이 부족
- AI는 반응형(위로 편지)이지, 감정 패턴 분석/예측 기능 없음
- 감정-활동-시간 상관관계 분석 미흡
- 한국어 감정 어휘 특화 아님
- 구독 가격($40/년)에 대한 "비싸다" 불만 존재

**마음날씨와의 차이:**
- 루빗: 습관 관리 중심 + 반응형 AI 위로
- 마음날씨: 감정 기록 중심 + 예측적 패턴 분석 + 셀프케어 행동 제안
- 루빗은 "감정을 기록하는 습관 앱", 마음날씨는 "감정을 분석하는 셀프케어 앱"

### 2-3. 라일리 하루 (Riley Haru) — **가장 유사한 신규 경쟁자** ⚠️

| 항목 | 내용 |
|------|------|
| 개발사 | 오티움 (대표 임태영) |
| 출시 | 2024년 12월 정식 출시 |
| 핵심 기술 | 독자 개발 소형 LLM + 감정 분석 알고리즘 |
| 팀 구성 | AI 개발자 15년 경력, 국문학 박사(감성 언어처리), UI/UX 디자이너, 마케터 |
| 대표 배경 | 미국 국립연구소 연구원, 전자공학 박사, 국내 대기업 IT 17년 경력 |

**핵심 기능:**
- 글, 사진, 음성 **멀티모달** 입력으로 감정 기록
- 자체 소형 LLM으로 텍스트/사진/음성 모두에서 감정 도출
- 감정 변화와 심리 패턴 시각화
- MBTI/심리테스트로 파악 불가한 개인 감정을 AI가 분석

**향후 계획:**
- 소셜미디어, 캘린더, 위치 데이터 자동 수집/분석
- 로컬 심리상담 센터, 명상 카페, 미술/음악치료 등 오프라인 멘탈케어 연결 플랫폼

**강점:**
- 자체 LLM으로 기술적 차별화
- 국문학 박사의 한국어 감성 언어처리 역량
- 멀티모달 입력 (텍스트+사진+음성)

**약점:**
- 2024년 12월 출시로 아직 초기 단계, 시장 검증 미완
- 팀 규모가 있어 비용 부담
- "셀프케어 행동 제안" 기능 부재 (분석 중심)

**마음날씨 대비 위협/기회:**
- ⚠️ **가장 직접적인 경쟁자**. AI 감정 분석에서 기술 우위 (자체 LLM vs SQL 통계)
- 그러나 "한국어 감정 어휘 체계"와 "실행 가능한 셀프케어 행동 제안"은 마음날씨의 차별점
- 라일리 하루가 "분석 중심"이라면, 마음날씨는 "행동 제안 중심"으로 차별화 가능

### 2-4. 데일리오 (Daylio) — **글로벌 1위 무드 트래커**

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 글로벌 1위 무드 트래커 |
| 지원 언어 | 30개+ (한국어 포함) |
| 주요 기능 | 기분+활동 연결 추적, 통계 분석 |

**핵심 기능:**
- 기분 5단계 + 활동 태그 기록
- 연간/월간/주간 무드 통계
- 활동-기분 상관관계 분석
- 목표 설정 및 달성 추적

**강점:**
- 감정-활동 상관관계 분석에서 가장 성숙한 기능
- 30개+ 언어 지원, 글로벌 사용자 기반
- 안정적인 앱 품질

**약점:**
- 한국어 감정 어휘 미반영 (번역 수준)
- 한국 사용자 반응: "감성 부족", "스마트폰 글쓰기 불편"
- 셀프케어 행동 제안 기능 없음
- 한국적 감성/디자인 부재

**마음날씨와의 차이:**
- 데일리오: 범용 글로벌 무드 트래커 (기분 5단계)
- 마음날씨: 한국어 특화 감정 기록 (30-40종 어휘) + 셀프케어 행동 제안
- 데일리오의 분석 기능은 강하나, 한국 문화 적합성에서 열위

| 항목 | 내용 |
|------|------|
| 개발사 | 올리브스톤랩 (소규모) |
| 출시 | 2019년 8월 |
| 평점 | 4.7점 / 리뷰 3,700+건 (한국) |

**핵심 기능:**
- 9가지 감정 아이콘으로 기록
- 캐릭터 위로 메시지 (감정별 맞춤)
- 다이어리 꾸미기 (사진, 스티커, 글씨체)
- 위젯, 알림, 잠금 기능

**강점:**
- 감성적 디자인, 위로 메시지
- 한국 감성에 맞는 UX
- 높은 평점 (4.7)

**약점:**
- 데이터 분석/인사이트 기능 약함
- 소규모 팀으로 업데이트 속도 제한
- 감정 이후 행동 제안 없음

**마음날씨 대비:** MOODA는 "감성적 기록" 특화, 마음날씨는 "분석+행동" 특화. 직접 경쟁보다는 보완적 관계.

### 2-6. 무디 (Moodee) — **감정 대응 퀘스트 앱** ⚠️

| 항목 | 내용 |
|------|------|
| 개발사 | 블루시그넘 (하루콩과 동일) |
| 출시 4개월 DL | 15만+ |
| 수상 | 구글플레이 '올해의 앱' 2023 (한국/홍콩) |
| 포지셔닝 | "감정 기록 → 행동 퀘스트" 연결 |

**핵심 기능:**
- 감정 입력 시 맞춤 '퀘스트' 추천 (**1,200개 모듈**)
- 감정 분석 보고서 (월간/연간)
- 가상 시나리오 대처 연습 (트레이닝)
- 스토리 모드 (동물 캐릭터)
- 비전 프로 음성 감정 분석

**마음날씨 대비:**
- ⚠️ **"감정→행동" 구조에서 가장 직접적인 기존 경쟁자**
- 무디: 규칙 기반 퀘스트 매칭 (1,200개 모듈) + 게이미피케이션
- 마음날씨: AI 패턴 분석 + 개인화된 셀프케어 행동 카드
- 무디는 "감정에 즉시 반응하는 퀘스트", 마음날씨는 "패턴을 분석해서 예측적으로 제안하는 행동"
- 블루시그넘이 하루콩(기록) + 무디(행동) + 라임AI(상담)의 생태계 구축 중. 이 생태계가 완성되면 마음날씨의 포지션 위협 가능

## 3. 경쟁 우위 분석

### 3-1. 마음날씨의 차별화 요소

| 차별점 | 강도 | 방어 가능성 | 설명 |
|--------|------|-----------|------|
| 한국어 감정 어휘 30-40종 | 중간 | 낮음 | 어휘 자체는 모방 용이. 그러나 "감정 분류 체계"를 최초로 구축하면 브랜드 인지도 확보 가능 |
| 감정-활동-시간 패턴 분석 | 중간~높음 | 중간 | SQL 통계 자체는 간단하나, 3개월+ 누적 데이터 기반 인사이트는 모방 어려움 |
| 셀프케어 행동 카드 | 중간 | 낮음 | 행동 제안 콘텐츠는 복제 가능. 사용자 피드백 기반 개인화가 핵심 방어선 |
| 기록→분석→행동 완결 루프 | 높음 | 중간 | 경쟁사 대부분 기록만 또는 상담만. 완결 루프는 통합적 가치 |
| 월 3,500원 가격대 | 중간 | 낮음 | 상담 앱 대비 1/10 가격. 가격 자체는 방어선이 아님 |

### 3-2. 경쟁사 대비 SWOT

**강점 (Strengths):**
- 한국어 감정 문화에 특화된 유일한 포지셔닝
- 기록→분석→행동의 완결된 가치 루프
- 기술적으로 단순 (SQL 통계), 외부 AI 의존 없음
- 필코노미/메타센싱 트렌드와 완벽한 정렬

**약점 (Weaknesses):**
- 1인 개발로 기능 확장/업데이트 속도 제한
- 캐릭터/감성 디자인 면에서 하루콩/MOODA에 열위 가능
- 초기 사용자 확보 마케팅 자원 부족
- 감정 기록 습관화의 근본적 어려움

**기회 (Opportunities):**
- 2026 필코노미 트렌드 탑승
