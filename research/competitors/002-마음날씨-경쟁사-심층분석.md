# 마음날씨 경쟁사 심층 분석

> 조사일: 2026-01-30

---

## 1. 경영진 요약

- **시장 규모**: AI 기반 정신건강 관리 시장은 2024년 12억 달러(약 1.65조원)에서 2032년 103억 달러(약 14.17조원)로 연평균 22.3% 성장 전망
- **경쟁 구도**: 한국 감정 기록 앱 시장은 하루콩(1,000만 DL), 무디(15만+ DL), 루빗(210만+ DL), 마보(60만+ DL), 트로스트(80만+ 고객) 등 다수의 플레이어가 존재
- **AI 도입 현황**: 대부분의 경쟁자가 AI 기능을 도입 중이나, "감정 패턴 분석 + 개인화된 셀프케어 행동 제안"을 하나의 앱에서 완결적으로 제공하는 경쟁자는 부재
- **핵심 기회**: 감정 기록(하루콩), 감정 퀘스트(무디), AI 위로(루빗), 명상(마보), 전문 상담(트로스트)이 각각 분리되어 있어, 이를 AI로 통합하는 포지셔닝 가능

## 2. 경쟁사별 심층 분석

### 2-1. 하루콩 (HaruBean)

**개발사**: 블루시그넘 (BlueSignum Corp.)
**출시**: 2021년
**누적 다운로드**: 1,000만+ (2025년 돌파, 236개국 서비스)
**해외 비중**: 약 80% (미국, 일본, 베트남 등 92개국에서 1만건+ 기록)

#### 핵심 기능

| 기능 | 상세 |
|------|------|
| **감정 기록** | 5가지 기분 콩(색상+표정)으로 하루 감정 선택, 아이콘 탭 방식으로 글 없이 기록 가능 |
| **활동 기록** | 알록달록한 아이콘으로 하루 활동 요약, 사진 첨부(최대 3장), 한 줄 메모 |
| **커스텀 기록 블록** | 기록 카테고리를 자유롭게 추가/삭제 가능, 카테고리 지속 업데이트 |
| **감정 통계** | 주/월 단위 활동별 기록 횟수 체크, 기분 흐름 시각화, 기분에 영향 주는 활동 분석 |
| **생활 연계 분석** | 운동 시간, 수면 시간, 월경 등과 감정 간의 관계 확인 가능 |
| **패턴 분석** | 기록 축적 시 감정과 생활 요소 간 상관관계 파악 (통계 기반, AI 아님) |
| **AI 기능** | **미도입** (2025년 기준 AI 기반 분석/제안 기능 없음) |

#### 가격/수익 모델

- **기본**: 무료 (광고 포함)
- **프리미엄**: 평생 이용권 "커피 한 잔 값" (약 3,000~5,000원대 추정)
- **수익원**: 프리미엄 구독료 + 캐릭터 테마 판매 (디즈니 콜라보 등)
- **투자**: 누적 21억원 유치, 월트디즈니코리아 콜라보 계약

#### 최신 동향 (2024~2025)

- 2024년 연간 감정 데이터 2,960만건 기록
- 2024년 운동 기록 등 신규 기능 추가 (웰니스 서비스 확장)
- 2025년 디즈니 테마 출시 예정
- 구글 스타트업 프로그램 선정
- 2021~2022년 2년 연속 구글플레이 '올해의 앱' 선정

#### 강점

1. **압도적 사용자 기반**: 1,000만 DL은 국내 감정 기록 앱 중 최대 규모
2. **극한의 간편성**: 글 없이 아이콘 탭만으로 기록 완료 → 지속률 높음
3. **캐릭터 IP**: '콩' 캐릭터가 브랜드 아이덴티티, 테마 판매로 수익 확장 용이
4. **글로벌 확장**: 해외 80%로 국내 의존도 낮음
5. **데이터 축적**: 2,960만건/년의 감정 데이터 보유 → 향후 AI 학습에 유리

#### 약점

1. **AI 기능 부재**: 패턴 분석이 단순 통계 수준, AI 기반 인사이트/제안 없음
2. **행동 제안 없음**: 감정 기록에 머물 뿐, "그래서 뭘 해야 하는지" 알려주지 않음
3. **기록 깊이 제한**: 아이콘 탭 방식은 간편하나 감정의 맥락/원인까지 파고들지 못함
4. **iPad/멀티디바이스 미지원**: 기기 간 수동 백업/복원 필요

#### 마음날씨 대비 분석

- **위협점**: 1,000만 사용자 기반의 네트워크 효과, 간편한 UX에 익숙한 사용자 이탈 저항
- **차별점**: 마음날씨의 AI 패턴 분석 + 맞춤 행동 제안은 하루콩이 제공하지 않는 핵심 가치
- **결론**: 하루콩은 "기록" 단계에 머물러 있고, 마음날씨는 "기록 → 분석 → 행동 제안"의 완결 루프를 제공. 직접 경쟁이라기보다 진화된 상위 카테고리

### 2-2. 무디 (Moodee)

**개발사**: 블루시그넘 (BlueSignum Corp.) ※ 하루콩과 동일 개발사
**출시**: 2023년 3월
**다운로드**: 출시 4개월 만에 15만건
**수상**: 2023년 구글플레이 '올해의 앱' (한국+홍콩 Fun 부문)
**해외 비중**: 약 80%

#### 핵심 기능

| 기능 | 상세 |
|------|------|
| **감정 기록** | 다양한 감정 태그로 현재 감정에 정확한 이름 붙이기 (Labeling) |
| **감정 통계** | 자주 기록한 감정, 시간대별 기분, 월간/연간 보고서 제공 |
| **퀘스트 추천** | 감정에 따라 맞춤 퀘스트(행동) 추천 → 기쁠 때/가라앉을 때 각각 다른 퀘스트 |
| **트레이닝** | 가상 시나리오에서 부정적 사고 패턴을 바꾸는 인지 훈련 (CBT 기반) |
| **스토리 모드** | 동물 친구들의 고민을 들어주며 자기 자신에게 필요한 말을 발견하는 콘텐츠 |
| **감정 저널** | 사적 감정 일기 + 보안 비밀번호 잠금 |
| **AI 기능** | 음성 감정 분석(리틀 무디/비전 프로), 1,200개 모듈 기반 맞춤 콘텐츠 (규칙 기반 추정) |

#### 가격/수익 모델

- **기본**: 무료 (제한적 기능)
- **프로(Pro)**: 인앱 구독 (정확한 가격 비공개, 추정 월 3,000~5,000원대)
  - 프로 전용: 무디와의 대화 월 30회, 트레이닝 기능 전면 이용
- **콘텐츠 품질**: 치료 효과 이론 기반 1,200개 모듈, 자문진 검수

#### 최신 동향

- 애플 비전 프로 전용 앱 '리틀 무디(Little Moodee)' 출시 → 음성 분석으로 감정 인식
- 보석(오브제) 형태로 감정을 시각화하는 공간 컴퓨팅 경험 제공
- 블루시그넘이 하루콩+무디 양축으로 운영하여 리소스 분산 가능성

#### 강점

1. **감정 → 행동 연결**: 감정 기록 후 바로 퀘스트(행동) 추천 → 마음날씨와 가장 유사한 구조
2. **심리학 이론 기반**: CBT(인지행동치료) 기반 트레이닝, 자문진 검수 콘텐츠
3. **퀘스트 다양성**: 1,200개 모듈로 감정/이슈별 맞춤 대응
4. **다중 접근**: 퀘스트 + 트레이닝 + 스토리 모드로 다양한 경로 제공
5. **구글플레이 '올해의 앱'**: 검증된 시장 적합성

#### 약점

1. **AI 깊이 부족**: 퀘스트 추천이 규칙 기반(태그 매칭)으로 추정, 개인화 수준 제한적
2. **패턴 분석 한계**: 통계 보고서는 제공하나 AI가 장기 패턴을 발견하고 예측하는 기능은 부재
3. **사용자 규모**: 15만 DL로 하루콩 대비 현저히 적음 (같은 개발사임에도)
4. **콘텐츠 중심**: 실시간 AI 대화보다 사전 제작 콘텐츠 소비에 가까운 구조
5. **프로 가격 장벽**: 무료 체험 옵션 미흡하다는 사용자 불만

#### 마음날씨 대비 분석

- **위협점**: "감정 기록 → 행동 퀘스트"라는 핵심 구조가 마음날씨와 가장 유사. 가장 직접적인 경쟁자
- **차별점**: 마음날씨는 AI가 장기 패턴을 분석하여 개인화된 제안을 하고, 무디는 규칙 기반 콘텐츠 매칭
- **결론**: 무디가 가장 가까운 직접 경쟁자. 마음날씨는 "AI 개인화"와 "장기 패턴 예측"에서 차별화 필요

### 2-3. 마보 (Mabo)

**개발사**: 마보 (대표: 유정은)
**출시**: 2016년 베타, 2018년 법인 전환
**누적 다운로드**: 60만+ (2023년 기준), 36만+ 가입
**포지셔닝**: 국내 최초 디지털 명상 앱

#### 핵심 기능

| 기능 | 상세 |
|------|------|
| **명상 콘텐츠** | 전문가 직접 제작 800+ 콘텐츠, 매주 2회 신규 업데이트 |
| **수면 명상** | 숙면 유도 전문 프로그램 |
| **명상 챌린지** | 하루 세 번 그룹 명상 기능 |
| **명상 음악** | 다양한 사운드스케이프 제공 |
| **마음일기** | 51만건+ 한국인 실제 마음일기 텍스트 데이터 보유 |
| **AI 기능** | AI 웰니스 코치 개발 중, LG 씽큐온 IoT 연동 (2025년) |

#### 가격/수익 모델

- **무료 체험**: 7일 무료
- **구독 가격**:
  - 1개월: 3,300~3,900원
  - 3개월: 9,900원
  - 6개월: 15,000원
  - 12개월: 27,000원
- **구독 전환율**: 8% (글로벌 평균 4% 대비 2배)
- **B2B**: 기업 EAP, LG 씽큐온 연동, 브랜디드 콘텐츠

#### AI 관련 최신 동향 (2025년)

1. **LG 씽큐온 연동**: AI 홈 허브와 협업, 생활 패턴 분석 → 취침 시 자동 수면 명상 실행, 음성 명령으로 명상 시작 + 에어컨/조명 자동 제어
2. **2026 마보백서 발행** (2025.11):
   - "AI가 만든 혼돈, 멘탈 피트니스가 필요하다" 주제
   - 51만건+ 한국인 마음일기 데이터를 AI 학습 자산으로 활용 방향 제시
   - **AI 웰니스 코치** + **B2B 관리자 대시보드** 개발 방향 발표
   - 선제적/예측적 멘탈 피트니스 인프라 구축 추진

#### 강점

1. **전문성**: 국내 최초 명상 앱, 정신과 전문의/상담사도 사용
2. **콘텐츠 품질**: 전문가 제작 800+ 콘텐츠, 매주 2회 업데이트
3. **데이터 자산**: 51만건 한국인 마음일기 → AI 학습 데이터로 활용 가능
4. **높은 전환율**: 8% 구독 전환율 (업계 2배)
5. **B2B 확장**: LG, 기업 EAP 등 B2B 채널 다각화

#### 약점

1. **명상 특화**: 감정 기록/분석보다 명상 콘텐츠 소비가 핵심
2. **AI 미완성**: AI 웰니스 코치는 아직 개발/계획 단계 (2025.11 백서 발표 시점)
3. **제한적 감정 분석**: 마음일기는 있으나 실시간 감정 패턴 분석 기능 부재
4. **행동 제안 없음**: 명상 이외의 셀프케어 행동 제안 미제공
5. **규모 한계**: 60만 DL로 하루콩(1,000만) 대비 소규모

#### 마음날씨 대비 분석

- **위협점**: 51만건 마음일기 데이터 기반 AI 웰니스 코치 개발이 실현되면 직접 경쟁자로 부상
- **차별점**: 마보는 "명상 앱 + AI 확장"이고, 마음날씨는 "감정 기록 → AI 분석 → 셀프케어 제안"이 핵심
- **결론**: 현재는 간접 경쟁. AI 웰니스 코치 출시 시 부분적 영역 겹침 가능성. 마보가 명상 중심인 반면 마음날씨는 일상 행동 제안 전반을 커버하여 차별화 가능

### 2-4. 루빗 (Roubit)

**개발사**: 주식회사 루빗 (대표: 이준영)
**설립**: 2021년 7월
**누적 다운로드**: 210만+
**타겟**: Z세대 비율 70%
**직원**: 9명 (2025년 기준)

#### 핵심 기능

| 기능 | 상세 |
|------|------|
| **습관/루틴 관리** | 반복 루틴 + 일회성 태스크 통합 관리, 요일별/카테고리별 세분화 |
| **감정 관리** | 감정 일기 작성 기능 |
| **AI 위로의 편지** | ChatGPT 기반, 감정일기 분석 후 맞춤 위로 편지 전달 |
| **AI 습관코치 챗봇** | 토끼 캐릭터 '에디'가 습관 조언 + 응원/위로 채팅 |
| **게이미피케이션** | 아기 루빗 캐릭터 육성, 루틴 완료 시 당근 획득 → 방 꾸미기 |
| **NFC 알람** | NFC 태그를 통한 강제 기록 시스템 |
| **위젯** | 홈 화면 위젯으로 할 일/투두 확인 |
| **친구 기능** | 친구 방 방문 및 응원 기능 |

#### 가격/수익 모델

- **기본**: 무료 (최대 8개 루틴)
- **Roubit Ultimate (프리미엄)**:
  - 연간 약 $40 (약 5.2만원) 기준
  - 무제한 루틴, 프리미엄 전용 테마
  - 40% 할인 프로모션 수시 진행
- **수익원**: 구독 + 인앱 구매 ($0.99~)
- **수상**: 2021년 구글플레이 '올해를 빛낸 자기계발 앱', 앱스토어 '오늘의 앱'

#### AI 기능 상세

- **AI 위로의 편지**: 사용자의 감정일기 내용을 ChatGPT가 분석 → 구체적인 위로 편지 작성. 조언뿐 아니라 친근한 톤으로 친구와 대화하는 듯한 느낌
- **AI 습관코치 챗봇 '에디'**: 사용자의 습관 데이터 기반 조언, 자유 채팅(응원/위로) 가능
- **기술 고도화**: 기존 시퀀셜 모델 + 생성형 AI 앙상블 구조로 진화 중
- **과학적 근거**: 행동활성화 치료(BA) 이론 기반 → 우울증 극복 목적

#### 강점

1. **습관 + 감정 통합**: 습관 관리와 감정 관리를 하나의 앱에서 제공
2. **ChatGPT 연동**: AI 위로 편지 + 습관코치 챗봇으로 실질적 AI 활용
3. **게이미피케이션**: 캐릭터 육성 + 보상 시스템으로 지속 사용 유도, Z세대 소구
4. **과학적 근거**: BA(행동활성화 치료) 이론 기반 설계
5. **사용자 규모**: 210만 DL, Z세대 70%로 명확한 타겟

#### 약점

1. **감정 분석 깊이 부족**: AI가 감정일기에 반응형 위로만 제공, 장기 패턴 분석/예측 없음
2. **습관 관리 중심**: 감정은 부가 기능이지 핵심이 아님
3. **AI 개인화 한계**: ChatGPT API 호출 수준으로, 사용자 누적 데이터 기반 개인화 부족 추정
4. **캐주얼 포지셔닝**: 게이미피케이션 중심이라 "진지한 감정 관리" 이미지와 거리
5. **프리미엄 가격**: 연 $40는 국내 경쟁 대비 높은 편

#### 마음날씨 대비 분석

- **위협점**: ChatGPT 기반 AI 감정 반응과 습관+감정 통합이 마음날씨와 부분 겹침
- **차별점**: 루빗은 "습관 관리 앱 + 감정 부가기능"이고, 마음날씨는 "감정 분석 중심 + AI 행동 제안"
- **결론**: 부분 경쟁. 루빗의 AI는 반응형(위로 편지)이고, 마음날씨의 AI는 분석형(패턴 발견 + 예측적 제안). 접근 방식이 근본적으로 다름

### 2-5. 트로스트 (Trost)

**개발사**: 휴마트컴퍼니
**포지셔닝**: 국내 1위 비대면 심리상담 앱
**누적 고객**: 80만+
**핵심 경쟁력**: 전문 상담사 네트워크 + AI 챗봇

#### 핵심 기능

| 기능 | 상세 |
|------|------|
| **AI 챗봇 '티티'** | 감정일기, 마음정리보고서, 감정스캐너 제공. 감정을 8개로 세분화, 16개 축으로 구분 |
| **감정 한 컷** | 오늘의 기분을 사진처럼 찍고 스티커+텍스트로 꾸미는 감정 기록 기능 |
| **전문 심리상담** | 1:1 텍스트테라피, 전화상담, 화상상담, 대면상담 예약 |
| **명상 & ASMR** | 기분별 사운드테라피, 볼륨 개별 조절 |
| **심리테스트** | 우울증, 불안, 자존감, 직무스트레스 등 전문 테스트 |
| **커뮤니티** | 고민 나누기, 코치/유저 공감과 응원 (2025년 신규) |
| **오늘의 명언** | 매일 격려 메시지 전달 (2025년 신규) |

#### 가격/수익 모델

- **무료 기능**: AI 챗봇 티티, 감정 한 컷, 심리테스트, 사운드테라피 (일부)
- **유료 상담**:
  - 텍스트테라피: 3만~4만원대/회
  - 전화상담: 약 5만원/회
  - 오프라인 대비 90% 저렴 (오프라인: 50분 10만원~)
- **기업용 EAP**: 구독형 패키지 (무제한 전문 심리상담, 월 단위 계약)
- **수익 구조**: 상담권 건별 판매 + 기업 EAP 구독

#### AI 챗봇 '티티' 상세

- **감정 분석 엔진**: 전산심리치료(Computational Psychotherapy) 기술 기반
- **감정 세분화**: 8개 감정 카테고리, 16개 축으로 정밀 구분
- **마음정리보고서**: 대화를 통해 감정 상태를 정리하여 보고서 형태로 제공
- **감정스캐너**: AI가 현재 감정 상태를 스캔하여 진단
- **NLP/ML 기술**: 자연어처리와 기계학습 모델을 활용한 감정 분석 솔루션 연구

#### 최신 동향 (2025년)

- 홈 화면 대규모 리뉴얼: 감정 중심으로 재구성, 필요한 콘텐츠를 한눈에 확인
- 커뮤니티 오픈: 고민 공유 + 코치/유저 공감 공간
- '오늘의 명언/성경 말씀' 신규 기능 추가
- 2025년 3월 5일 최신 업데이트 확인

#### 강점

1. **전문 상담 연결**: AI 자가관리에서 전문 상담으로 자연스러운 업셀링 경로
2. **감정 분석 기술력**: NLP/ML 기반 8x16 감정 분류 체계는 가장 정교
3. **브랜드 신뢰도**: "국내 1위 심리상담 앱" 포지셔닝, 80만 고객
4. **익명성 보장**: 완전 익명 이용 가능 → 심리 서비스 진입 장벽 낮춤
5. **B2B/B2C 양면**: 기업 EAP + 개인 상담으로 수익 다각화

#### 약점

1. **AI 챗봇 한계**: 티티는 감정 진단/보고서 수준이고, 지속적 패턴 분석이나 예측 기능은 약함
2. **상담 중심 모델**: 핵심 수익이 유료 상담에 의존 → 일상적 자가관리보다 "문제 발생 시" 대응적
3. **일상 행동 제안 부족**: 명상/ASMR 외 구체적인 셀프케어 행동 제안 미흡
4. **가격 부담**: 상담 1회 3~5만원은 일상적 감정관리 도구치고 고가
5. **감정 기록 깊이**: '감정 한 컷'은 시각적이나 데이터 분석에 활용하기 어려운 구조

#### 마음날씨 대비 분석

- **위협점**: 감정 분석 기술력(NLP/ML)이 가장 앞서 있으며, "AI 자가관리 → 전문 상담"이라는 완결적 경로 보유
- **차별점**: 트로스트는 "문제 해결형(상담 연결)", 마음날씨는 "일상 예방형(패턴 분석 + 셀프케어)"
- **결론**: 포지셔닝이 다름. 트로스트는 "심리적 어려움이 있을 때" 찾는 앱, 마음날씨는 "매일 쓰는 감정 관리 앱". 오히려 상호보완적 관계

## 5. 마음날씨의 전략적 차별화 포인트

### 5-1. 경쟁사가 채우지 못한 갭

1. **패턴 예측**: 모든 경쟁자가 "과거 데이터 통계"만 제공. "앞으로의 감정 변화 예측"을 제공하는 앱 부재
2. **행동 제안의 개인화**: 무디의 퀘스트가 가장 근접하나 규칙 기반. AI가 개인 이력 기반으로 "이 사용자에게 가장 효과적이었던 행동"을 추천하는 앱 부재
3. **기록→분석→행동의 완결 루프**: 기록(하루콩), 행동(무디), 상담(트로스트)이 각각 분리. 한 앱에서 완결하는 서비스 부재
4. **트리거 인식**: "어떤 상황에서 어떤 감정이 반복되는지" AI가 자동 발견하여 알려주는 기능 부재

### 5-2. 권장 포지셔닝

```
"마음날씨 = 하루콩의 간편 기록 + 무디의 행동 제안 + AI 패턴 예측의 통합"
```

- 하루콩처럼 간편하게 기록하되
- 무디처럼 구체적 행동을 제안하되
- AI가 장기 패턴을 분석하고 예측적 인사이트를 제공

### 5-3. 진입 전략 우선순위

1. **즉시 차별화 가능**: AI 감정 패턴 분석 (경쟁자 전원 미제공)
2. **중기 차별화**: 개인화된 행동 효과 추적 ("이 행동 후 기분이 좋아진 비율 82%")
3. **장기 차별화**: 예측적 알림 ("내일 월요일, 지난 4주간 월요일 아침 스트레스가 높았습니다")
